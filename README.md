# Smart RFID Attendance System

## Overview

A complete RFID-based attendance management system built using Arduino RC522 RFID module, Flask backend, SQLite database, and a responsive web dashboard.

## Features

- Real-time RFID card scanning
- Automated attendance logging
- Student registration system
- Admin dashboard
- Attendance analytics
- SQLite database integration
- ESP32/Arduino support
- CSV export

## Hardware Used

- Arduino Uno
- RC522 RFID Reader
- ESP32-CAM
- I2C LCD Display
- DFPlayer Mini
- Buzzer
- LEDs

## Software Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## System Architecture

RFID Card
↓
Arduino + RC522
↓
Serial Communication
↓
Flask Backend
↓
SQLite Database
↓
Web Dashboard
